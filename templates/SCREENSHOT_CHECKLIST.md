# Screenshot Checklist

- [ ] Use real product screens from the current version.
- [ ] Remove personal data, credentials, phone numbers, and private customer information.
- [ ] Capture desktop at approximately 1440 × 900.
- [ ] Capture mobile screens at consistent device proportions.
- [ ] Include the home screen and the strongest business workflow.
- [ ] Include an admin/dashboard screen when it is a core feature.
- [ ] Use descriptive filenames such as `desktop-inventory.png`.
- [ ] Add meaningful alt text in README.
- [ ] Do not stretch low-resolution images.
- [ ] Do not use fake browser chrome or invented UI.
- [ ] Keep language and RTL direction visible.
- [ ] Optimize files before committing.
