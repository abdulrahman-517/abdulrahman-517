# GitHub Metadata Checklist

## Description formula

`An Arabic-first [product type] for [target user] with [two or three verified capabilities].`

## Topics

Choose only accurate topics from:

- Framework: `nextjs`, `react`, `fastapi`
- Language: `typescript`, `javascript`, `python`
- Styling: `tailwindcss`
- Data: `postgresql`, `supabase`, `redis`
- Product: `dashboard`, `saas`, `pwa`, `marketplace`
- Experience: `arabic`, `rtl`
- Domain: `real-estate`, `automotive`, `transportation`, `search`

## Required review

- [ ] Name is concise and descriptive.
- [ ] Description is not “test,” “final,” or “my project.”
- [ ] Website URL has been opened and verified.
- [ ] Topics match code in the repository.
- [ ] Social preview is uploaded through GitHub settings.
- [ ] Visibility is intentional.
- [ ] License is present and accurate.
- [ ] Default branch is correct.
